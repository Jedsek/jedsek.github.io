先贴代码:

(define (A x y) (cond 
  ((= y 0) 0)
  ((= x 0) (* 2 y))
  ((= y 1) 2)
  (else (A (- x 1)
           (A x (- y 1))))))

(A 1 10)
; 1024

(A 2 4)
; 65536

(A 3 3)
; 65536

(define (f n) (A 0 n))
(define (g n) (A 1 n))
(define (h n) (A 2 n))

你可以手动展开诸如 (A 1 10) 这样的代码, 当然, 也可以选择使用 racket/trace 中的 trace 来跟踪递归情况:

(require racket/trace)
(trace A)
(A 1 10)
